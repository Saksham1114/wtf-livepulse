# 🏋️ WTF LivePulse Dashboard Demo

A real-time gym analytics dashboard that tracks live occupancy, revenue, and anomaly detection across multiple gym locations.

---

## 🚀 Features

- 📊 Live Gym Occupancy Tracking
- 💰 Revenue Analytics (last 30 days)
- 🚨 Real-time Anomaly Detection
- 🔄 WebSocket-based Live Updates
- 🧠 Background Simulation Engine
- 🐳 Fully Dockerized Setup

---

## 🏗️ Tech Stack

### Frontend
- React (Vite)
- Tailwind CSS
- WebSockets

### Backend
- Node.js
- Express.js
- PostgreSQL
- WebSocket Server

### Infrastructure
- Docker & Docker Compose

---

docker compose up --build
